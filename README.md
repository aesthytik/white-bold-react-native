# React Native Theme

Super simple theme for react native.

## Stack

- [React Native](https://facebook.github.io/react-native/)

## To use in your React Native Project

```
yarn add ReactNativeTheme
```

## Setup

Run:

```
yarn install
```

## Development

To start storybook

```
yarn run storybook
```

Start App

```
yarn run <platform>
```
